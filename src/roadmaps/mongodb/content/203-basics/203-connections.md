# Connection

There are various way to connect to your MongoDB database deployment. Here are some of the most common ways you can connect to your Atlas cluster / MongoDB deployment.

* [Connect via Your Application](https://www.mongodb.com/docs/atlas/driver-connection/)
* [Connect via Compass](https://www.mongodb.com/docs/atlas/compass-connection/)
* [Connect via mongosh](https://www.mongodb.com/docs/atlas/mongo-shell-connection/)
* [Browse Data via the Atlas UI](https://www.mongodb.com/docs/atlas/atlas-ui/)
* [Manage connections via AWS Lambda](https://www.mongodb.com/docs/atlas/manage-connections-aws-lambda/)
* [Connect via VS Code](https://www.mongodb.com/docs/atlas/mongodb-for-vscode/#std-label-mongodb-for-vscode)
* [Connect via BI Connector for Atlas](https://www.mongodb.com/docs/atlas/bi-connection/)
* [Connecting to a MongoDB Database](https://learn.mongodb.com/courses/connecting-to-a-mongodb-database)
