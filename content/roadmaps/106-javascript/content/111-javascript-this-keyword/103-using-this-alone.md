# Using this alone

The keyword `this` when used alone refers to the global object.

*Note: in a browser window the global object is the `window` object.*

**Example**

```js
let x = this;
console.log(x);
```

**Output**

```js
Window {parent: Window, opener: null, top: Window, length: 0, frames: Window, …}
```
