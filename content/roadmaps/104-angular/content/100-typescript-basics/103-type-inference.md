# Type inference
In TypeScript, there are several places where `type inference` is used to provide type information when there is no explicit type annotation. The type of the x variable is inferred to be number. This kind of inference takes place when initializing variables and members, setting parameter default values, and determining function return types. For example, `let x: number`.
