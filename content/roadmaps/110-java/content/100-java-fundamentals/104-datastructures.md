# Datastructures

As the name indicates itself, a **Data Structure** is a way of organizing the data in the **memory** so that it can be used efficiently.

Here is the list of some of the common types of data structures in Java:

- Array
- Linked List
- Stack
- Queue
- Binary Tree
- Binary Search Tree
- Heap
- Hashing
- Graph
